## Premiers Pas

### 1. Connexion
1. Accédez à l'application via votre navigateur
2. Entrez le courriel de votre entreprise
3. Saisissez votre nom d'utilisateur et mot de passe
4. Cliquez sur "Se connecter"

### 2. Navigation
- Utilisez le **menu latéral gauche** pour naviguer entre les modules
- Le **tableau de bord** affiche vos KPIs en temps réel
- Les **notifications** apparaissent en haut de l'écran

### 3. Aide
- Chaque module dispose d'une section d'aide contextuelle
- L'**Assistant IA** peut répondre à vos questions en temps réel
- Contactez le support via la messagerie intégrée
